# 串口助手

多功能串口调试上位机，基于 Electron + Web Serial API 构建。

## 功能特性

- 多串口管理 — 支持同时打开和管理多个串口设备
- 串口配置 — 波特率（2400~921600）、数据位（5~8）、停止位、校验位
- 数据收发 — 支持 HEX / ASCII 模式收发
- 自动发送 — 可配置间隔的定时自动发送
- 波形显示 — 实时数据波形（ASCII数值 / HEX / Int16 / Float32）
- 数据保存 — 将接收数据导出为文本文件
- USB 设备检测 — 自动识别已连接的 USB 设备信息
- 热插拔检测 — 实时监测串口/USB设备的连接与断开

## 快速开始

下载 Releases 中的便携版，解压后双击 `启动串口调试助手.bat` 即可运行。

### 开发环境

```bash
npm install
npm run dev    # 开发模式
npm run dist   # 打包发行版
```

## 技术栈

- 框架: Electron 28
- 串口通信: Web Serial API（无需原生驱动）
- 图表: Chart.js 4.4
- 界面: 深色主题，三栏响应式布局
